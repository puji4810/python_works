see.dash_init()[源代码]¶

初始化 Dash 应用程序并设置包含各种可视化的布局。 返回：

app (Dash): Dash 应用程序实例。 df (DataFrame): 包含 CPU 数据的 DataFrame。

布局包括：

  • 标题头。

  • 显示 CPU 数据的数据表。

  • 显示 CPU 综合排名的柱状图。

  • 比较单核、多核和游戏排名的散点图。

数据处理：

  • 获取 CPU 列表和 JSON 格式的数据。

  • 解析和处理 CPU 数据。

  • 确保数据类型正确，并用 0 替换 NaN 值。

  • 创建 DataFrame 的副本，并为可视化目的替换特定排名。
